# Automotive-Sales-Analysis Dashboard
Automotive sales data analysis using SQL, Python, and Power BI

## Objective  
Analyze automotive sales data to understand revenue trends, regional performance, and top-selling models, helping businesses make data-driven decisions.

## Tools Used  
- SQL  
- Excel  
- Power BI  

## Key Tasks  
- Cleaned and transformed raw sales data using SQL and Excel  
- Built an interactive Power BI dashboard with filters for model, manufacturer, and dealership  
- Visualized KPIs such as revenue trends, regional sales, and top-performing products  

## Key Insights  
- Identified high-performing car models and regions  
- Observed seasonal trends in vehicle sales  
- Highlighted dealerships with the highest revenue contribution  

## Output  
An interactive Power BI dashboard that helps stakeholders track sales performance and optimize business strategy.
